# Wildermyth Legacy

[Site](https://manapart.github.io/wildermyth-legacy/) that allows you to view your Wildermyth Legacy without having to start up the game.

## Deploying
```
cp -r ../wildermyth-legacy/build/distributions/* . && rm characters.zip
```
